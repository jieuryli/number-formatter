Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @jieuryli/number-formatter`

## Usage

    var numFormatter = require('@jieuryli/number-formatter');

    var formattedNum = numFormatter(35667);


  Output should be `35,667`


## Tests
